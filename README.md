# File Sorter (Python)

Простой Python-скрипт для автоматической сортировки файлов в папке по расширениям.

## Возможности
- Сортировка файлов по типам (`.txt`, `.jpg`, `.pdf` и т.д.)
- Автоматическое создание папок
- Поддержка Linux и Windows
- Работает из терминала

## Использование

python3 sorter.py /path/to/folder
